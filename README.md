# Blog-Webstie

A Website that lets you write personal blogs and acts as an ongoing onlie diary

## Getting Started

- Install The required npm packages in the root directory
- run The local mongo database
```
$ mongod
``` 
- run app.js
```
$ node app.js
``` 
- launch the site on localhost:3000 

### Built With

- Node.Js
- MongoDB
- npm packages
  - express
  - ejs
  - body-parser
  - mongoose
